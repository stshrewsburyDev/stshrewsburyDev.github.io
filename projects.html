Yuri PyBot

Project website

A simple and free to use multipurpose Discord bot.

This bots functionality varies from image generation based on user input to searching the internet from within Discord. This bot also allows users to play various games making use of Discord's chat functionality.

PIL-Tools

GitHub repo PyPi page Documentation

A Python module that acts as an extension to the image generation module PIL (Pillow).

The module adds a range of pre-made functions that allows the user to easily manipulate images in certain ways. For example rendering rainbow text or applying a green screen effect to an image.
